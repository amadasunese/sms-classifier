About SMS Spam Detection

Welcome to SMS Spam Detection, a powerful tool designed to identify and filter out spam messages from your SMS inbox. This application leverages machine learning techniques to analyze the content of SMS messages, distinguishing between legitimate messages and potential spam.

Key Features:

  • Accurate Classification: The model achieves an accuracy of 98.39%, ensuring reliable spam detection.
  • User-Friendly Interface: This web-based interface allows users to easily input SMS messages and receive instant spam detection results.
  • Data Privacy: Your privacy is guaranteed. The application does not store any personal or message data entered into the application.

How It Works:

The SMS Spam Detection model was trained using a dataset of labeled SMS messages. It preprocesses the text, extracts features, and utilizes a Multinomial Naive Bayes classifier for accurate classification. Users can simply input an SMS message, and the application will provide feedback on whether the message is identified as spam or not.
